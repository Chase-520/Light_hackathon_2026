# Oneiros

Oneiros is a static HTML/CSS/JavaScript prototype for recording dreams, lighting dream-map landmarks, and replaying generated dream videos.

## Files

- `index.html` - app entry page
- `styles.css` - visual design
- `app.js` - app interaction logic
- `assets/` - dream replay videos
- `render.yaml` - Render static site configuration

## Render

Use these settings if configuring manually:

- Build Command: `echo No build needed`
- Publish Directory: `.`
